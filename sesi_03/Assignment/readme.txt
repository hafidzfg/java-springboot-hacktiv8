Panduan Penggunaan Aplikasi : 

Folder ini berisi sekumpulan latihan mengenai operator di Java.

Untuk menjalankan tiap latihan, eksekusi command berikut:
    java <NamaFile>
Contoh untuk menjalankan latihan "IncrementDecrement"
    java IncrementDecrement

1. Latihan 1: LuasLingkaran
Berfungsi untuk menghitung luas lingkaran berdasarkan jari-jari lingkaran yang diketahui.
    a. Jalankan: java LuasLingkaran
    b. Masukkan nilai jari-jari lingkaran dalam centimeter

2. Latihan 2: Aritmatika
Berisi latihan penggunaan operator aritmatika pada java
    java Aritmatika

3. Latihan 3: IncrementDecrement
Berisi latihan penggunaan increment dan decrement
    java IncrementDecrement

4. Latihan 4: PerbandinganTrueFalse
Berisi latihan penggunaan operator logika
    java PerbandinganTrueFalse

5. Latihan 5: PenjumlahanXY
Berisi latihan penggunaan operator aritmatika dan operator logika
    java PenjumlahanXY
